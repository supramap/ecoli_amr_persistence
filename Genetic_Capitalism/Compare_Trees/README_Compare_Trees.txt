README for Compare_Trees directory.

Includes an R script to compare two phylogenetic trees using .newick files.

The following two files are identical.
Also included are two test .newick files to verify the script:
- PDG000000004.1024.reference_target.tree.newick.txt
  Also found under supramap/ecoli_amr_persistence/Genetic_Capitalism/TNT/TNTFileGenerator/
  The original tree file for this project.
- PDG000000004.1024.newick.txt
  A supplementary file obtained from NCBI to verify the above file
